🔗 Connect Admin to Your Handled Website

Since your admin panel and main site live in different GitHub repos, they need a way to share data. Choose a connection method below.

📦 Option A: JSONBin.io (Free Cloud Storage)

JSONBin is a free REST API that stores JSON. Both your main site and this admin panel read/write from the same "bin". No server needed.

  1. Go to jsonbin.io and create a free account.
  2. Click Create Bin and paste this starter JSON:
    {"users":{},"requests":[]}
  3. Copy the Bin ID (looks like 64a1b2c3e4f5…) and your Master Key (starts with $2b$…).
  4. Paste them below and click Save & Test Connection.
  5. Add the same Bin ID and Master Key to your main site's store.js (instructions in the code snippet below).
